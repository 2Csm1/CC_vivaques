- RSA numerical (key generation, encryption, decryption)

#### Step 4: Safe Question Selection Every Paper

**For Q1 (pick 4 of 5):** Always pick:
1. Block cipher modes question
2. MD5/SHA question
3. Digital signature OR security mechanisms
4. One more from whatever appears

**For Q2–Q6 (attempt only 3):** Always attempt:
